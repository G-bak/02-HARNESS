- `--prompt` + `@path`: Gemini 0.40.0에서 positional prompt 충돌로 거부됨.
- `cmd.exe /c gemini.cmd ...`: quoted prompt argv가 깨져 positional prompt 충돌 발생.
- `--sandbox`: 로컬에서 장시간 block됨.
- 최종 경로:
  - prompt payload는 `validator-b-prompt-{N}.txt` artifact에 저장
  - `--prompt`에는 일반 파일 경로 지시문만 전달
  - read-only는 `--approval-mode plan` 사용
  - Windows `.cmd`는 PowerShell argv-array invocation 사용
