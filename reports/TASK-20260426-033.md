## 요청 요약

- 재진입 게이트와 `TASK_CREATED` 우선순위 충돌을 정리했다.
- Analyst의 제한적 감사 Shell 권한을 공식화했다.
- Tier 1 브랜치 예외와 non-git 작업공간 예외를 명시했다.
- `task_status`, `notification_status`, `severity`를 분리했다.
- Task Spec 원본 저장 위치를 `tasks/specs/TASK-{ID}.json` 또는 `TASK_CREATED.details.spec`으로 정의했다.
- 품질 점수 저장 위치를 `logs/quality-scores.jsonl`로 정의했다.
- 세션 로그의 인라인 보고 원문 보존을 강제 규칙으로 추가했다.
