## 설계 판단

- Validator 실행과 main merge는 분리했다. 검증은 기본 `read-only` sandbox이고, merge는 별도 절차에서만 수행한다.
- Codex CLI `--json` stdout은 이벤트 로그로 저장하고, `--output-last-message` 파일을 정규화된 결과로 파싱한다.
- Resource Failure는 Validator FAIL로 취급하지 않는다. `RESOURCE_FAILURE` + `HOLD`로 남겨 main 머지를 차단한다.
- 동일 실패 fingerprint는 `evidence_type + location + normalized description`으로 판정한다.
