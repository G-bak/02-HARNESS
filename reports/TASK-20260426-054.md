- `scripts/validate-ledger.mjs`, `scripts/check-completion-gates.mjs`

  - `details.tier` 값도 `Tier1|Tier2|Tier3`로 정규화해 게이트 판단에 반영한다.
- `logs/tasks/TASK-20260426-042`~`052`
  - 기존 이벤트는 수정하지 않고 `CORRECTION` 이벤트를 append했다.
  - session artifact 누락 또는 legacy `SUCCESS` 상태를 보정했다.
- `scripts/notify-slack.mjs`, `notification-policy.md`, `TASK-EXAMPLE.md`, `slack-webhook-setup.md`
  - 내부 이벤트 의미를 `notification_status` 중심으로 통일했다.
  - CLI는 기존 `--status` 호환을 유지하되 새 예시는 `--notification-status`를 사용한다.
- `task-lifecycle.md`, `task-spec.md`, `output-formats.md`
  - Task Spec 원본 저장을 필수 기준으로 통일했다.
  - Task Spec status enum에 `PARTIAL`을 추가했다.
  - Analyst 원장 예시를 표준 event schema 중심으로 정리했다.
- `CURRENT_STATE.md`
  - 변경된 권위 문서 버전을 갱신했다.
