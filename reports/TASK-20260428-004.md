## 질문 답변 정리

4번 질문의 답은 “맞지만 그것만으로는 부족하다”이다. `claude -p ... --output-format json`은 CLI stdout을 JSON으로 받는 장치다. 여기에 `--json-schema`를 추가해서 Generator 출력 형식을 요구하고, wrapper가 결과를 한 번 더 파싱해 필수 필드를 검증하도록 했다.

5번은 Markdown handoff 문제다. Markdown은 사람이 읽기에는 좋지만 `refs.spec`, `refs.ledger`, `allowed_context`, `forbidden_context`, 출력 경로를 기계적으로 강제하기 어렵다. 그래서 wrapper 실행은 JSON handoff만 허용하도록 바꿨다.

6번 Validator/retry 자동화는 이번에 구현하지 않았다. 사용자가 적용 전 설명을 요청했기 때문에 최종 응답에서 설계를 먼저 제시한다.
