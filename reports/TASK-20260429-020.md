## 2. 에이전트별 활동 이력

### 🟡 Researcher (Codex CLI, external_cli_fallback)

- **실제 호출 여부:** ✅
- **호출 횟수:** 2 (1차: capability probe / 2차: 풀 파이프라인 실행)
- **런타임 메타:** codex.cmd / cumulative duration ~5분 / sandbox=workspace-write / approval=never / ephemeral=true
- **Cost:** 약 467,747 input tokens (361,472 cached), 6,539 output + 1,928 reasoning tokens. 추정 ~$0.30–0.60 (Codex 구독 사용).
- **Execution mode:** `external_cli_fallback` (사용자 명시 요청에 따라)

**실제 한 일 (substance):**

Researcher는 두 단계로 작업했다. **1단계는 capability probe**로, 단순 "빨간 사과 이미지 생성해봐" 요청에 대해 Codex가 자기 skill 디렉토리(`C:\Users\admin\.codex\skills\.system\imagegen\SKILL.md`)를 PowerShell로 직접 읽어서 자기 능력을 자료 기반으로 보고했다. 이 단계에서 발견된 핵심: **Codex CLI에는 `image_gen.imagegen` built-in skill이 있고, default 모델은 `gpt-image-2` (최대 4K), OPENAI_API_KEY 불필요**. 1차 시도가 read-only sandbox라 워크스페이스 저장은 실패했지만 이미지 생성 자체는 작동 확인.

**2단계 풀 파이프라인 실행** (sandbox=workspace-write로 격상):

먼저 LandingHub 컨텍스트 확보 — `https://landinghub-api.lanfinghub-kr.workers.dev/showcase` 라이브 fetch는 **Codex sandbox의 외부 HTTP 차단으로 실패**. 그래서 로컬 README.md를 직접 읽어 "국내 랜딩페이지 쇼케이스 + 뉴스레터 관리 시스템 / Cloudflare Workers + D1 + Resend / Vanilla JS + Tailwind" 라는 핵심을 파악. 이 fetch 실패를 **숨기지 않고 confidence_rationale에 명시**한 점이 evidence 디스플린의 작동을 보여준다.

다음 ChatGPT Image 최신 뉴스 컨텍스트를 위해 web_search 3회 시도 → 6개 sources 확보:
- OpenAI 4o Image Generation 출시 (2025-03-25)
- OpenAI gpt-image-1 API 출시 (2025-04-23)
- OpenAI ChatGPT Images 새 모델 (2025-12-16)
- Axios "ChatGPT Images major update" (2026-04-21) — wider aspect ratios, thinking mode, text rendering, ads/posters/mock-ups 용도

Research Summary (`research-summary.json`)는 schema 그대로 작성: research_question / scope.include & exclude / sources[] (각각 title, url, relevance, snippet) / confidence: MEDIUM / confidence_rationale (라이브 fetch 실패 사유 명시) / unverified_claims / log[].

Ad poster prompt 작성은 `references/sample-prompts.md` 의 ads-marketing 슬러그를 따라:
- Use case: ads-marketing
- Asset type: social-feed poster, portrait 1024x1536
- Style: 한국 스타트업 미감, 흰색/검정/일렉트릭 블루/그린 액센트
- Verbatim Korean text 5줄: "LandingHub" / "국내 랜딩페이지를 한눈에" / "좋은 사례를 모으고, 뉴스레터 구독으로 연결하세요" / "쇼케이스 · 구독 · 발송 관리" / "지금 확인하기"
- Constraints: 가짜 회사 로고/브랜드 금지, QR 코드 금지, 가짜 URL 금지, OAuth/API key/env 절대 금지

이미지 생성은 **built-in image_gen tool 호출** (gpt-image-2 default)로 1024x1536 portrait PNG 출력 → `$CODEX_HOME/generated_images/` 임시 저장 → PowerShell `Copy-Item`으로 `download/TASK-20260429-020-landinghub-poster/poster.png`로 이동. 마지막에 `Add-Type -AssemblyName System.Drawing`으로 image dimension/format 검증까지 자체 수행.

**경계 준수:**
