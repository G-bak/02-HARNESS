## 인라인 보고
- 문서 구조를 바꾸는 작업이므로, 먼저 작업 원장을 만들고 규칙을 함께 정리하겠다고 알림.
- 세션 종료 시 사용자에게 보낸 최종 요약과 같은 결론으로 맞추는 규칙을 문서에 추가하겠다고 알림.
- `work-history-policy`와 `task-lifecycle`을 읽기 쉬운 한국어로 다시 작성한다고 알림.
- 세션 로그는 `작업 요약` 대신 `인라인 보고` 중심으로 바꾸겠다고 알림.

## 결정 사항
- 세션 종료 시 남기는 요약은 사용자에게 보낸 최종 설명과 같은 결론이어야 한다.
- 작업 중에는 짧은 인라인 업데이트를 남기고, 세션 종료 시 이를 정리해 세션 로그에 반영한다.
- 이 규칙을 `work-history-policy`와 `task-lifecycle`에 반영했다.
