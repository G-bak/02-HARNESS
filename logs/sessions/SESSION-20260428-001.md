- 본 작업은 Tier 1로 분류한다. 단일 가이드 파일 신규 생성, 권위 문서·제품 코드 변경 없음, 즉시 되돌리기 가능.

- **가이드 위치 정정 (사용자 지시):** 최초 `guides/`에 생성했으나 기존 운영 가이드(`mcp-setup.md` 등)와 동일한 디렉토리인 `docs/guides/`로 이동, 임시로 만든 루트 `guides/` 디렉토리는 삭제. 운영 가이드(Operational guide) 클래스로 둔다.

- **`docs/agents/analyst.md` 포함 (사용자 확인):** 해당 파일의 마크다운 포맷 변경은 사용자가 직접 수행한 것으로, 본 Task 커밋에 함께 포함한다.
